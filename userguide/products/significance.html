Types

significance
Small product used to manually set the significance value of an event. This can be used to promote or demote any event. The current threshold for the web to consider an event “significant” is ≥600.
Significance is calculated from properties of the origin, losspager and dyfi products. It is calculated from the following multi-step formula:
magnitude_significance
= (100/6.5) * magnitude2
pager_significance
=  2000 if red   1000 if orange  500 if yellow
dyfi_significance
= MIN(num_Resp, 1000) * maxmmi * 0.10
significance
= MAX(magnitude_significance, pager_significance) + dyfi_significance
