Introduction

The content of a simple text-type product is displayed directly on the event page. Text may be plain text or HTML markup. However, markup is not validated nor reviewed; product producers sending this type of product must take care to ensure the text content is appropriate and accessible.

Types

general-text
general-text products are displayed on the Overview page for the event.
scitech-text
scitech-text products are displayed on the Technical page for the event.
impact-text
impact-text products are displayed on the Impact page for the event.
deleted-text
deleted-text products are displayed when an event has been deleted.

Properties

content-path (proposed)
Default inline. Specify to override the default content path, for example to render content from index.html

Contents

inline
The text to display on the event page.
OTHER

When using HTML markup, relative URLs to other attached product contents may be used. For example, an image named "my_image.png" can be attached to the product, and referenced from the inline markup with a relative path like 

<img src="my_image.png" alt="My Image"/>
